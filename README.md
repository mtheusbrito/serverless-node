# Node.js com serverless framework

#### Requisitos:

NodeJS >=16.18.1



#### Como baixar:
```

// Clonando repositório
git clone git@github.com:mtheusbrito/serverless-node.git 

// Acessar diretório
cd serverless-node/ 

// Instalar dependências
npm install

```
#### Executando o projeto:

```bash
npm run dev
```

`http://localhost:3000`.
