# Node.js com serverless framework

#### Requisitos:

- NodeJS >=18.18.2
- Serverless CLI


#### Como baixar:
```

// Clonando repositório
git clone git@github.com:mtheusbrito/serverless-node.git 

// Acessar diretório
cd serverless-node/ 

// Instalar dependências
npm install

```
#### Executando o projeto:

```bash
npm run dev
```

`http://localhost:3000`.


#### Executando o serverless localmente:

```bash
sls offline
```
